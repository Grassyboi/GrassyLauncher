# Grassy Launcher

Grassy Launcher is a free, light python program to run cracked Minecraft.

# Requirements:

  •Python
  
  •customtkinter
  
 •minecraft-launcher-lib

  # How to install requirements:

•Download Python from the offical website or another VERY trusted source

•After that press (Windows+R), then type in "cmd" and press enter

•Then paste in this command:

    pip install customtkinter minecraft-launcher-lib

# Tip:

  **If you are not supposed to have any form of Minecraft or cracked Minecraft, I would recomend putting the file in a random folder**
